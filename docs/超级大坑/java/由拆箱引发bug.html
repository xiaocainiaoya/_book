由拆箱引发的NPE

由拆箱引发bug
public static void main(String[] args) {
  int a = 0;
  Integer b = null;
  System.out.println(a == b); // Exception in thread "main" java.lang.NullPointerException
}

由于b是包装类型，而a是基本数据类型，在比较时对b进行拆箱，所以会通过((Integer)null).intValue()进行拆箱，从而引发NPE。
